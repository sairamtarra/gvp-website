# GVP Website

This is a simple static website for **Gayatri Vidya Parishad College for Degree and PG Courses (Autonomous)**.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- Images stored in the `Gvp Images` folder

## How to Run

1. Clone or download this repository.
2. Open the file: `GVP WEBSITE.html` in any web browser (Chrome, Edge, etc).
3. Make sure the `Gvp Images` folder is in the same directory as the HTML file so the images load correctly.

## Features

- College header and contact info
- Navigation bar using Bootstrap 4
- Image carousel using Bootstrap 4
- Codes and important information boxes
